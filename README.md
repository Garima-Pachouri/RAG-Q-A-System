# RAG-Based Question Answering System

## Overview
This project demonstrates a basic idea of Retrieval-Augmented Generation (RAG), where relevant information is retrieved first and then used to generate answers.

## Tech Stack
- Python

## Features
- Simple document retrieval
- Context-based answering
- Demonstrates RAG concept

## How It Works
- User enters a query
- System searches matching documents
- Relevant information is returned as answer
